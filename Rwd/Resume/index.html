The main objective of this project is to identify the bag exacts location by using GSM and GPRS. By using mobile Application, it provides real-time tracking using a map. In addition to it, an RFID tag is also used to create a unique ID for the passengers and the notification sent to the user when an unauthorized person accesses the luggage.
